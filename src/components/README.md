# Components

React components for the field visualizer application.

## Structure

### Layout Components
- `ScenePanel.tsx` - 3D viewport container (Phase 1)
- `Sidebar.tsx` - Control panel with UI elements (Phase 1)

### 3D Scene Components
- `FieldVectors.tsx` - Field arrow visualization (Phase 6)
- `SourceSphere.tsx` - Individual source charge/mass rendering (Phase 4)
- `TestChargeSphere.tsx` - Test charge/mass rendering (Phase 4)
- `ForceArrow.tsx` - Force visualization on test charge (Phase 7)

### UI Components
- `FormulaPanel.tsx` - Formula display with live values (Phase 9)
- `ModeToggle.tsx` - Mode switcher (Phase 8)
- `SourceList.tsx` - Source management UI (Phase 8)
- `TestChargeControls.tsx` - Test charge controls (Phase 8)
- `PresetsPanel.tsx` - Scenario presets (Phase 11)
